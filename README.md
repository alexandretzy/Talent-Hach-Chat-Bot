# Talent Hatch AI Automation

## Overview
**Talent Hatch** is a company platform that automates HR tasks, including managing applicant resumes and providing a customized **AI Agent Chatbot**. Built using **React**, **Tailwind CSS**, **Shadcn UI**, and **Supabase**, this platform streamlines the hiring process by automating the collection, review, and analysis of resumes. It also includes an AI-powered chatbot that answers user inquiries, helping both applicants and HR teams manage tasks more efficiently.

The chatbot is designed to handle common HR-related questions, while the system automates resume review tasks, ensuring quick and accurate results for job applicants and recruiters alike.

## Features
- **Applicant Resume Management**: Automatically collects, reviews, and stores resumes based on job applications.
- **AI Agent Chatbot**: Provides real-time, contextually relevant answers to user inquiries about the company, hiring processes, and other HR-related questions.
- **Integration with Supabase**: Uses **Supabase** for data storage, such as resumes and applicant details.
- **Customizable HR Workflow**: Automates routine HR tasks, such as reviewing resumes and handling applicant queries, reducing manual effort and speeding up the hiring process.

## Prerequisites

Before running the project, ensure you have:

- **Git** installed
- **Node.js** (includes npm) installed
- **Supabase account** for backend services

## Steps to Run the Project

### 1. Clone the Repository
- Open GitBash Command, then run this code below:

```bash
git clone https://github.com/ZyroGlitch/Talent-Hatch.git
cd your-repo-name
```

### 2. Install Dependencies
- Once, you're now in your project directory run this code below:
```bash
npm install
```

### 3. Run the Development Server
- Run this code below to start running your React project in localhost.
```bash
npm run dev
```
